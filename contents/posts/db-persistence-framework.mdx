# DB

> 프레임 워크 선정을 위해 조사했던 것을 간략히 정리하였다.

객체와 관계형 데이터베이스 연결해야 한다.

JDBC를 이용할 수 있지만, 매우 복잡하기 때문에 Persistence framework를 이용한다.

- Persistence framework 종류
  - SQL 문장으로 직접: mybatis ⇒ sql mapper
  - 객체를 통해 간접적으로(ORM): hybernate

### mybatis

- 장점
  - sql 쿼리를 직접 작성하여 최적의 성능을 낼 수 있다.
  - 다양한 테이블을 조합할 수 있다.
- 단점
  - DB에 종속된 쿼리가 발생할 수 있다.
  - 쿼리를 직접 수정해야 한다.
  - 런타임에 오류를 확인할 수 없다.

### ORM

- 장점
  - 더 직관적
  - 객체 재사용 및 유지보수 편리
  - DBMS 종속성 줄어든다
- 단점: 쿼리를 직접 다루지 않아 완벽히 구현이 어렵다?
