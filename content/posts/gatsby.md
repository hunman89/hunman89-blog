# gatsby로 블로그 만들기

> 아무도 찾지않는 블로그
> gatsby를 이용해 리모델링 하자!

### 가입

Gatsby cloud에 가입하면 홈페이지 하나 무료로 deploy 해줌, 일단 가입하자!

### 시작

1. CLI 설치
   > 새로운 Gatsby사이트를 쉽게 만들기위해

```shell
npm install -g gatsby-cli
```

잘 됬는지 확인해보자

```shell
gatsby --version
```
